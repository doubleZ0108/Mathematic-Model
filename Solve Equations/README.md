## 一元高次方程

### 夹逼法

1. $f(a)*f(b)<0$，根在a，b中间
2. 取$m = (a+b)/2$，计算$f(m)$
3. 比较$f(m)$和$f(a)$，同号则$a = m$(根在b，m之间，a缩到m)，否则$b = m$
4. 当$|ab|<\Delta$时，结束，否则跳转到2

## 遗传算法寻优

求方程的零点

1. 给定一个初值$x_0$，计算$y_0$
2. 产生一个随机小数$\Delta$，再产生一个随机整数`sign`
   - sign是偶数，$x_1 = x_0 + \Delta$
   - else，$x_1 = x_0 - \Delta$
3. 计算$x_1$的函数值$y_1$
   - if y1 < y0，说明x1比x0优秀，保留它，跳转到【2】
